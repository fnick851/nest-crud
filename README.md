# nest.js CRUD app

## Features

- RESTful API

- Validation and error handling

- Authentication (JWT) and authorization

- PostgreSQL and TypeORM

- Logging

- Testing

## Installation

```bash
# install dependencies
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
